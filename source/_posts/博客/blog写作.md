## 标题

Markdown 支持六级标题，使用 `#` 符号表示：

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

**效果预览：**
- # 一级标题
- ## 二级标题
- ### 三级标题

## 引用

引用文本有两种方式：

### 方法一：直接缩进（使用 Tab）

```markdown
    这是引用内容（前面加4个空格或Tab）
```

### 方法二：使用引用符号

```markdown
> 这是引用内容
> 可以多行引用
```

**效果预览：**

> 这是引用内容
> 可以多行引用

## 列举

### 无序列表

使用 `*`、`-` 或 `+` 符号：

```markdown
* 项目一
* 项目二
  * 子项目一
  * 子项目二
```

**效果预览：**
* 项目一
* 项目二
  * 子项目一
  * 子项目二

### 有序列表

使用数字加 `.`：

```markdown
1. 第一项
2. 第二项
3. 第三项
```

**效果预览：**
1. 第一项
2. 第二项
3. 第三项

## 代码

### 行内代码

使用单个反引号包裹：`` `代码` ``

**示例：** 使用 `printf()` 函数输出

### 代码块

使用三个反引号包裹代码，可以指定语言：

````markdown
```c++
#include <iostream>
using namespace std;
```
````

**C++ 示例：**
```c++
#include <iostream>
using namespace std;
```

**Python 示例：**
```python
import os
import sys
```
